# Quran & Prayer App

تطبيق إسلامي متكامل صُمّم ليجمع بين الجمال، السرعة، والتفاعل الذكي في تجربة واحدة يومية.

## نبذة سريعة
- مواقيت صلاة دقيقة مع حساب الصلاة القادمة تلقائيًا.
- مصحف تفاعلي حديث بنمط صفحات مصحف مع أدوات حفظ ومراجعة.
- تخصيص متقدم للويدجت مع معاينات عملية على الشاشة الرئيسية.
- تنبيهات مرنة ومفصلة لكل صلاة وخيارات تذكير قبل الأذان.
- واجهة محسنة للشاشات الكبيرة والأجهزة القابلة للطي (Foldables).

## مميزات التطبيق
- `مواقيت الصلاة الذكية`
  - دعم طرق حساب متعددة والمذهب الفقهي.
  - عرض الصلاة القادمة والوقت المتبقي بشكل واضح.
  - التعامل الصحيح بعد العشاء بإظهار فجر اليوم التالي كصلاة قادمة.

- `نظام التنبيهات المتقدم`
  - تفعيل/تعطيل إشعارات كل صلاة بشكل مستقل.
  - تذكير قبل الأذان بفاصل زمني قابل للتخصيص.
  - خيارات الصوت والاهتزاز مع إدارة عملية للتنبيهات.

- `المصحف التفاعلي (Mushaf Pro)`
  - عرض صفحات مصحف مع تنقل سلس بين الصفحات.
  - حفظ فواصل الصفحات والآيات والعودة لها بسرعة.
  - الانتقال المباشر لأي صفحة.
  - وضعان مخصصان:
    - `وضع القراءة`: تجربة قراءة هادئة وواضحة.
    - `وضع الحفظ`: إخفاء/إظهار النص + اختبار سريع للمراجعة.
  - `Focus Mode`: جلسات تركيز 10/20/30 دقيقة مع مؤقت مباشر.

- `تخصيص الويدجت باحترافية`
  - خيارات ألوان وخطوط وأنماط متعددة.
  - إعدادات مستقلة لكل ويدجت.
  - معاينات تسهّل اختيار الويدجت المناسب قبل التطبيق.

- `أقسام إضافية مفيدة`
  - الأذكار.
  - السبحة.
  - القبلة + خريطة القبلة.
  - التقويم الهجري.
  - المناسبات الإسلامية.
  - أسماء الله الحسنى.
  - تتبع الصلاة الفائتة والزكاة.

- `تصميم وتجربة استخدام`
  - دعم اللغة العربية بشكل متقن.
  - تحسينات مرئية واسعة على الشاشات الكبيرة وأجهزة الفولد.
  - تجربة سلسة وسريعة عند فتح التطبيق والتنقل بين الصفحات.

## التقنية المستخدمة
- `Flutter` + `Dart`
- `Riverpod` لإدارة الحالة
- `Hive` للتخزين المحلي
- `Adhan` لحساب المواقيت
- `Flutter Local Notifications` للتنبيهات
- `Home Widget` للويدجت

## البناء والتشغيل
```bash
flutter pub get
flutter run
```

لبناء نسخة إصدار:
```bash
flutter build apk --release
```
